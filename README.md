# Hernoem

## Format

| Key  | Result to    | Specialties                       |
|------|--------------|-----------------------------------|
| `%a` | Artist       | Multiple would be joined with `,` |
| `%A` | Album artist | Multiple would be joined with `,` |
| `%t` | Title        |                                   |
| `%b` | Album        |                                   |
| `%y` | Release year |                                   |
| `%n` | Track number | Track 1 would result to 01        |
| `%g` | Genre        |                                   |
| `%c` | Composer     |                                   |
| `%d` | Disc         |                                   |
